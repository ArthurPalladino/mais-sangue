# Mais Sangue

Mais Sangue foi um projeto desenvolvido no primeiro semestre de 2024, é um aplicativo de banco de dados para doadores sanguíneos (pacientes), integrado a um sensor biométrico para garantir a veracidade do doador.

No segundo semestre, durante um projeto de 7 dias da disciplina de Desenvolvimento Mobile, decidi recriar o Mais Sangue para dispositivos móveis utilizando React Native, e ir além do solicitado criando uma API em .NET. Esse projeto marcou meu primeiro contato com .NET e React Native em um prazo bastante apertado. Apesar dos desafios e erros de projeto durante o desenvolvimento, o trabalho foi essencial para meu aprendizado, especialmente em relação à integração entre front-end e back-end.

---

## Tecnologias utilizadas

- Front-end Mobile: React Native  
- Back-end / API: .NET  
- Banco de dados: MySql
